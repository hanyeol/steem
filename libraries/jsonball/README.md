# Jsonball Library

High-performance JSON parsing and generation optimized for blockchain data.

## Overview

The `jsonball` library provides:
- Fast JSON serialization
- Efficient parsing
- Optimized for blockchain operations
- Memory-efficient representation

## Features

- **Performance**: Optimized for high-frequency JSON operations
- **Compatibility**: Works with FC reflection system
- **Efficiency**: Minimal memory allocations
- **Integration**: Seamless with Steem types

## Usage

Used throughout the codebase for:
- API request/response handling
- Configuration parsing
- State serialization
- RPC communication

## Build

```bash
cd build
make steem_jsonball
```

## Dependencies

- `fc` - Variant and reflection support
