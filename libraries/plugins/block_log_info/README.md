# Block Log Info Plugin

Provides information about block log files.

## Overview

The block_log_info plugin:
- Analyzes block log structure
- Reports block log statistics
- Validates block log integrity
- Provides diagnostic information

## Configuration

```ini
plugin = block_log_info
```

## Features

- **Log Analysis**: Block log file information
- **Statistics**: Block count, size, range
- **Validation**: Integrity checking
- **Diagnostics**: Troubleshooting support

## Use Cases

- Block log validation
- Storage management
- Troubleshooting
- Migration planning

## Dependencies

- `chain` - Block log access
