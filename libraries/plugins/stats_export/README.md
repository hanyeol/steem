# Stats Export Plugin

Exports blockchain statistics to external systems.

## Overview

The stats_export plugin:
- Collects blockchain metrics
- Exports statistical data
- Supports monitoring integration
- Tracks network health

## Configuration

```ini
plugin = stats_export

# Export configuration
# stats-export-endpoint = "endpoint"
```

## Features

- **Metrics Collection**: Blockchain statistics
- **Export**: Push to external systems
- **Real-time**: Continuous updates
- **Customizable**: Configurable metrics

## Use Cases

- Monitoring dashboards
- Performance analysis
- Network health tracking
- Operational metrics

## Dependencies

- `chain` - Blockchain data
